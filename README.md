# 灵桥 Creative Hub

这是把 `素材抓取`、`KV生成`、`表情包` 三个独立项目统一到一个平台入口中的第一版整合层。

## 当前架构

- `frontend/`: 平台壳层，提供 Dashboard、项目中心、平台设置和模块入口
- `platform_api/`: 统一项目索引、模块状态、平台配置中心
- `素材抓取/content-marketing/`: 原内容营销模块，保持独立
- `KV生成/`: 原 KV 生成模块，保持独立
- `表情包/`: 原表情包模块，保持独立

## 端口约定

- Hub 前端: `5180`
- Platform API: `8100`
- 素材抓取后端/前端: `8001 / 5174`
- KV 生成后端/前端: `8002 / 5175`
- 表情包后端/前端: `8003 / 5176`

## 一键启动

在根目录运行：

```powershell
.\start.ps1
```

或双击：

```bat
start.bat
```

## 第一版目标

- 一个统一平台入口
- 三个模块可同时运行
- 统一项目中心索引
- 统一平台设置和模块状态页
- 保留原业务模块的独立运行能力
